# Confidence Interval Method for tr_performance Objects

Confidence Interval Method for tr_performance Objects

## Usage

``` r
# S3 method for class 'tr_performance'
confint(object, parm = NULL, level = NULL, ...)
```

## Arguments

- object:

  A `tr_performance` object.

- parm:

  Parameter (ignored, only one parameter).

- level:

  Confidence level (default uses object's conf_level).

- ...:

  Additional arguments (ignored).

## Value

A matrix with confidence interval bounds.
